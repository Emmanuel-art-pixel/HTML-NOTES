HTML

HTML is basically HyperText Markup Language. It is a basic language used to structure web pages.

Common HTML Tags

  1. <html>
  2. <head>
  3. <title>
  4. <body>
  5. <p> (paragraph)
  6. <a> (link)
  7. <div> (container)

HTML Attributes and Formatting

Attributes give extra information to tags
  • href – link destination
  • src – image source
  • alt – image description
  • id – unique identifier
  • style – inline styling

Forms and Inputs

The most useful component of a form is the input tag, which creates a text field where users can enter data.
